# renminbi_color_style Asset Manifest

Curated SVG assets for `renminbi_color_style`. These are original, code-native SVG interpretations for personal design work. They are **not** scans of legal tender and must not be used to recreate printable currency, official banknote layouts, portraits, seals, serial numbers, watermarks, security threads, or anti-counterfeit marks.

## Source And Provenance

- Bundled files in this folder are manually authored SVG assets inspired by safe financial layout primitives and RMB color families. They are not direct downloads or copies.
- Project note: current project is personal-use oriented; still record source inspiration and avoid counterfeit-like reproduction.

## Usage Rules

- Preserve intrinsic aspect ratio.
- Keep financial linework below content hierarchy; do not place behind dense table text, chart labels, or legal/compliance copy.
- Combine with `transparent_textures` only when the style file enables the texture token and contrast remains readable.
- Prefer one major asset per slide/section plus smaller repeated accents; avoid making every surface look like a voucher.
- If a user supplies real currency imagery, use only for documentary/editorial context and never as a UI background or printable design template.
