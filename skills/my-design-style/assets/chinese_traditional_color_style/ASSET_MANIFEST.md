# chinese_traditional_color_style Asset Manifest

Curated SVG assets for `chinese_traditional_color_style`. These assets support restrained new-Chinese / guofeng layouts with borders, stripes, side bands, and placeholder imagery while avoiding overloaded antique decoration.

## Source And Provenance

- Base SVG files in this folder are manually authored assets inspired by common Chinese border, lattice, ink-wash placeholder, and pattern-band roles.
- Project note: current project is personal-use oriented; keep provenance notes so future public/commercial reuse can re-check licensing.

## Usage Rules

- Preserve intrinsic aspect ratio unless the asset is explicitly a stripe or band and the crop does not deform motif geometry.
- Use at low density; one border/frame plus one small stripe is usually enough.
- Do not stack all symbols at once. Traditional color, whitespace, hierarchy, and rhythm carry the style first.
- Replace `03_landscape_placeholder.svg` with user-provided or project-specific imagery when the final content requires a real object/photo.
- Do not borrow SEU identity assets or RMB financial assets into this style unless the user explicitly requests a hybrid and one primary style remains clear.
