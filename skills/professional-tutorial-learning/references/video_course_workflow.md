# Video Course Workflow

Use this workflow for video courses, lecture recordings, timestamped transcripts, or playlists.

## Workflow

1. Prefer transcript or timestamped outline over raw video links.
2. Capture title, URL, instructor/publisher when known, module, timestamp range, and transcript confidence.
3. Split by timestamps, slide changes, topic transitions, or exercise demonstrations.
4. Classify demonstrations as supporting examples unless they encode a reusable procedure.
5. Compress filler speech, repeated setup, and non-instructional banter.
6. Generate a markdown lesson with timestamp references where available.
7. Add practice and review prompts that do not require rewatching the whole video.

## Failure Modes

- If no transcript or outline is available, ask for one or provide a watching plan instead of pretending to have processed the video.
- If timestamps are uncertain, mark them as inferred.
